JsonSchemaEntities 

trait JsonSchemaEntities extends EndpointsWithCustomErrors with algebra.JsonSchemaEntities with JsonSchemas

Interpreter for algebra.JsonEntities that uses Play JSON play.api.libs.json.Reads to decode JSON entities in HTTP requests, and play.api.libs.json.Writes to build JSON entities in HTTP responses.
